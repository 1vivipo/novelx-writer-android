# NovelX Writer - AI小说写作助手

一款基于 AI 的小说自动写作工具，支持安卓原生应用。

## 功能特点

- 🎨 **原生安卓应用** - 使用 Kotlin + Jetpack Compose 开发
- 🤖 **AI 自动写作** - 智能生成世界观、人物、情节、章节
- 📚 **项目管理** - 支持多项目管理，实时查看进度
- 📖 **章节阅读** - 内置阅读器，随时阅读已生成的章节
- ☁️ **云端同步** - 数据存储在云端，多设备同步

## 技术栈

- **语言**: Kotlin
- **UI框架**: Jetpack Compose
- **架构**: MVVM
- **网络**: OkHttp + Gson

# 云端自主写小说工具 - 项目说明

## 系统概述

这是一个完整的云端自主写小说工具，实现了以下核心功能：

1. **10个智能体协作**：世界观师、大纲架构师、人物设计师、情节设计师、章纲规划师、正文撰写师、文风润色师、连续性检查员、查重审核员、质量总监
2. **云端自主运行**：用户输入提示词后，系统自动完成设定生成和逐章写作
3. **并发写作**：最多支持3本小说同时写作
4. **去AI味**：自动检测和去除AI写作痕迹
5. **查重审核**：章节内和跨章节查重
6. **连续性检查**：确保故事设定一致性

## 写作风格

- **文风**：参考鲁迅先生，语言精炼，白描手法，讽刺幽默
- **情节设计**：参考冰临神下，环环相扣，反转自然，节奏张弛有度
- **人物刻画**：参考《六爻》，性格鲜明，感情线自然，单男主单女主

## 10个智能体详解

### 1. 世界观师 (WorldviewArchitect)
- 构建小说的世界观、背景设定、地理环境
- 设计力量体系、社会结构、主要势力
- 创建世界地图和历史背景

### 2. 大纲架构师 (OutlineArchitect)
- 设计故事的整体架构和主线发展
- 规划故事弧线、核心冲突、转折点
- 设计高潮和结局

### 3. 人物设计师 (CharacterDesigner)
- 设计主角、女主、反派、配角
- 创建人物关系网络
- 规划人物成长线

### 4. 情节设计师 (PlotDesigner)
- 设计关键场景和冲突
- 规划伏笔和转折
- 设计战斗场景和感情场景

### 5. 章纲规划师 (ChapterPlanner)
- 将大纲细化为每章规划
- 分配情节和节奏
- 设计章节钩子

### 6. 正文撰写师 (ChapterWriter)
- 根据设定和章纲撰写正文
- 保持文风一致
- 确保与上文连贯

### 7. 文风润色师 (StylePolisher)
- 润色正文，接近鲁迅文风
- 去除AI味
- 提升文学性

### 8. 连续性检查员 (ContinuityChecker)
- 检查与设定的一致性
- 检查与上文的连贯性
- 检查人物性格一致性

### 9. 查重审核员 (DuplicateChecker)
- 检查章节内重复
- 检查跨章节重复
- 识别重复句式

### 10. 质量总监 (QualityDirector)
- 综合评估章节质量
- 决定是否通过或重写
- 提供改进建议

## 写作流程

### 阶段1：设定生成
1. 世界观师生成世界观设定
2. 大纲架构师生成故事大纲
3. 人物设计师生成人物设定
4. 情节设计师生成情节设计
5. 章纲规划师生成章节细纲

### 阶段2：逐章写作
对于每一章：
1. 正文撰写师根据设定和章纲写作
2. 文风润色师润色正文
3. 连续性检查员检查一致性
4. 查重审核员检查重复
5. 质量总监审核决定是否通过
6. 如不通过，重写（最多3次）
7. 通过后保存为txt文件

## 去AI味机制

系统自动检测和去除以下AI痕迹：
- "仿佛"、"似乎"、"宛如"等AI常用词
- "那一刻"、"这一刻"等时间标记
- "心中涌起一股..."类心理描写
- "眼中闪过一丝..."类描写
- 过于工整的排比句式
- 重复的句式和描写

## 查重机制

- 章节内重复率 < 5%
- 跨章节重复率 < 3%
- 与设定冲突 = 0

## 使用方法

1. 打开网页界面
2. 在"创建项目"页面输入提示词和章节数
3. 点击"开始云端自主写作"
4. 系统自动开始生成设定和写作
5. 在"监控面板"查看进度
6. 在"项目详情"查看设定和章节
